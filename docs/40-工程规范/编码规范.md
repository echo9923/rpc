# 项目编码规范

## 一、基础规范

### 1.1 文件组织

| 规范项 | 具体要求 |
|--------|----------|
| 头文件保护 | 使用 `#pragma once` |
| 编码格式 | UTF-8，无 BOM |
| C++ 标准 | C++20（`CMAKE_CXX_STANDARD 20`） |

### 1.2 缩进与格式

| 规范项 | 具体要求 |
|--------|----------|
| 缩进 | 4 个空格，不使用 Tab |
| 函数定义大括号 | 换行风格 |
| 控制语句大括号 | 紧凑风格，与控制语句保持在同一行 |
| 指针声明 | 使用 `Type *name` |
| 空指针 | 使用 `nullptr` |

### 1.3 类型别名

| 规范项 | 具体要求 |
|--------|----------|
| 别名方式 | 使用 `using`，禁止使用 `typedef` |
| 命名规范 | 使用 PascalCase |
| 示例 | `using Socket = int;`、`using ReadCallback = std::function<void()>;` |

## 二、命名规范

### 2.1 类型命名

| 类型 | 规范 | 示例 |
|------|------|------|
| 类名 | PascalCase | `TcpServer`、`TcpConnection`、`Logger` |
| 枚举名 | PascalCase | `LogLevel`、`ErrorCode` |
| 枚举值 | PascalCase 或全大写；同一个枚举内保持一致 | `LogLevel::INFO`、`ErrorCode::Success` |
| 结构体 | PascalCase | `SocketOption`、`RequestContext` |
| 类型别名 | PascalCase | `using Socket = int;`、`using Buffer = std::string;` |

### 2.2 变量命名

| 变量类型 | 规范 | 示例 |
|----------|------|------|
| 成员变量 | `m_` 前缀 + lowerCamelCase | `m_localAddress`、`m_listenFd`、`m_fd` |
| 全局变量 | lowerCamelCase；常量按常量命名规则处理 | `serverName`、`listenPort` |
| 普通局部变量 | lowerCamelCase | `clientFd`、`readBytes`、`peerAddress` |
| 常量 | `k` 前缀 + PascalCase | `kInvalidSocket`、`kMaxBufferSize` |

### 2.3 方法与函数命名

| 类型 | 规范 | 示例 |
|------|------|------|
| 获取类方法 | `get` 前缀 + lowerCamelCase | `getLocalAddress()`、`getReadableBytes()` |
| 判断类方法 | `is`/`has` 前缀 + lowerCamelCase | `isConnected()`、`hasData()` |
| 普通方法 | lowerCamelCase | `closeConnection()`、`registerToReactor()` |
| 普通函数 | lowerCamelCase | `createSocket()`、`setNonBlock()` |
| 私有工具函数 | lowerCamelCase | `acceptLoop()`、`readData()`、`writeData()` |

### 2.4 文件命名

| 规范项 | 具体要求 | 示例 |
|--------|----------|------|
| 文件名 | 全英文小写 | `tcpserver.cc`、`tcpconnection.h` |
| 头文件与实现文件 | 同一模块的头文件与实现文件命名保持一致 | `tcpserver.h` + `tcpserver.cc` |
| 测试入口文件 | 放在 `testcases` 目录，文件名保持英文小写；新增测试优先使用 `test_*.cc` | `test_tcp_echo_server.cc` |

## 三、Include 规范

### 3.1 Include 顺序

头文件按以下顺序分组，组之间保留一个空行：

1. 本地头文件：`#include "net/tcpserver.h"`
2. 第三方头文件：Boost、nlohmann-json 等
3. 标准库头文件：`#include <string>`、`#include <memory>`

### 3.2 示例

```cpp
#include "net/tcpconnection.h"

#include <string>
```
